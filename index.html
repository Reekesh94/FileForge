✦ About FileForge

Built for Everyone.
Free & Private.

FileForge is a fully browser-based file processing suite. Every operation runs directly in your browser using JavaScript Canvas API and PDF-lib — your files never leave your device and are never uploaded to any server.

Whether you need to compress an image for a website, merge PDFs for a report, or convert between document formats — FileForge gives you professional-grade tools without subscriptions, logins, or upload limits.

Built with vanilla HTML, CSS, and JavaScript split into modular files for easy maintenance. Zero heavyweight frameworks. Leverages pdf-lib for PDF manipulation, the Canvas 2D API for image processing, and JSZip for bulk downloads.
